# Portfolio

An animated, modern portfolio built with Next.js, Framer Motion, and Tailwind CSS.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- Dark theme with gradient accents
- Framer Motion animations
- Responsive design
- Contact form
- Timeline section

## Customization

1. Replace "Your Name" and placeholder content in components
2. Add your images to `/public/images`
3. Update social links in Hero and Footer
4. Add your projects to Projects.jsx
5. Update timeline in Timeline.jsx

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- React Icons
